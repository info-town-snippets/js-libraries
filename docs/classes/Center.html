中央配置

CSSの絶対配置を使い親要素の中央へ配置します。

Parameters:

  • target JQuery

    中央配置するjQueryオブジェクトです。

  • parent JQuery

    親要素のjQueryオブジェクトです。

  • [offset] Object optional

    センターからのオフセット値です。

    • [top] Number optional

      Y座標のオフセット値を指定します。

    • [left] Number optional

      X座標のオフセット値を指定します。

Returns:

Object:

設定したtop, leftの値を返します。
